TROUBLESHOOTING
───────────────────────────────────────────────────────────────────────────────

  → Check: netstat -ano | findstr ":8000"
  → Restart: Kill uvicorn process and restart

  → Check: netstat -ano | findstr ":11434"
  → Fix: Run "ollama serve" in separate terminal

ChromaDB Issues
  → Reset: Delete data\chroma_db directory
  → Reinit: Run scripts/init_db.py

Database Locked
  → Cause: Multiple FastAPI instances
  → Fix: Ensure only one backend instance running
