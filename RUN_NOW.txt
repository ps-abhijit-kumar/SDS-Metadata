                        HOW TO RUN THE PLATFORM
================================================================================

STEP 1: Open PowerShell Terminal 1 (Backend)
──────────────────────────────────────────────────────────────────────────────

cd c:\Coding\Projects\SDS-Metadata-Extractor

.venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000 --workers 1 --timeout-keep-alive 120

Wait for:
  INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)


STEP 2: Open PowerShell Terminal 2 (Frontend)
──────────────────────────────────────────────────────────────────────────────

cd c:\Coding\Projects\SDS-Metadata-Extractor

.venv\Scripts\python.exe -m streamlit run frontend/app.py --server.port 8501 --server.address 127.0.0.1

Wait for:
  You can now view your Streamlit app in your browser.
  URL: http://127.0.0.1:8501


STEP 3: Open Web Browser
──────────────────────────────────────────────────────────────────────────────

http://127.0.0.1:8501


STEP 4: Use the Platform
──────────────────────────────────────────────────────────────────────────────

1. Click "Upload SDS PDF documents"
2. Select a PDF file
3. Click "Extract"
4. Wait 20-90 seconds
5. View results:
   - File ID

================================================================================
                    TROUBLESHOOTING
================================================================================

Backend won't start:
  → Check Ollama is running: ollama serve
  → Check port 8000 is free: netstat -ano | findstr ":8000"
  → Check logs: type logs/app.log

Extraction times out:
  → Normal: First file takes 60-90 seconds (model warmup)
  → Subsequent files are faster
  → Monitor logs for progress

Connection reset:
  → Restart backend terminal
  → Check logs/app.log for errors
  → Ensure Ollama models are downloaded

Database locked:
  → Restart backend (SQLite WAL recovery)
  → Only one backend instance should run
