   - Language
   - Jurisdiction

================================================================================
                       WHAT'S HAPPENING
================================================================================

Backend is processing documents through a 10-stage RAG pipeline:
  1. PDF Text Extraction
  2. Text Normalization
  3. SDS Section Detection
  4. Semantic Chunking
  5. Embedding Generation
  6. Vector Store Ingestion
  7. Semantic Retrieval
  8. LLM Query (Qwen3:8B)
  9. Metadata Parsing
  10. Result Storage

All processing is 100% LOCAL - no cloud APIs or internet required.

================================================================================
                         TROUBLESHOOTING
================================================================================

Backend crashed?
  → Restart backend terminal
  → Check logs/app.log for error details
  
Extraction slow?
  → Normal: First file takes 60-90 seconds (model warmup)
  → Subsequent files are faster
  
Connection refused?
  → Ensure both terminals are still running
  → Backend may need restart
  
Ollama not connected?
  → Run: ollama serve
  → Ensure ollama models are downloaded
